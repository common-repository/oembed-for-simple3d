워드프레스 사이트에 3D모델을 삽입하세요

== Description ==

oembed plugin for Simple3D.net
Simple3D에 업로드한 3D모델을 워드프레스 사이트에 삽입할 수 있습니다.   
 

= Simple3D =

[Simple3D](http://www.simple3d.net)는 3D 파일을 온라인에 저장하는 공간입니다.
심플3D에 보관된 3D파일은 온라인은 통한 공유는 물론 제공되는 다양한 효과를 추가 더욱 가치있는 콘텐츠를 다시 태어날 수 있습니다.   


= Simple3D WordPress Plugin =

심플3D 뷰어는 모든 웹 페이지에 삽입될 수 있습니다. 심플3D 플러그인을 설치하고, [simple3d.net](http://www.simple3d.net)에서 3D모델의 url을 복사하여여 포스트에 붙여넣기 만으로 여러분의 워드프레스의 사이트에 3D모델을 삽입할 수 있습니다.    


== Installation ==

= 설치 =
1. 워드프레스 대시보드에서 플러그인 메뉴를 선택한 후 플러그인 추가 버튼을 누릅니다.
2. "Simple3D oEmbed"를 검색합니다.
3. 설치 버튼을 눌러 플러그인을 설치합니다.
4. 활성화 버튼을 눌러 플러그인을 활성화시킵니다.

= 업데이트 =
* 워드프레스 자동 업데이트를 활용하면 Simple3D을 최신 상태로 유지할 수 있습니다. 단, 만약을 대비하여 사이트를 자주 백업하는 것을 잊지 마십시요.

= 최소 요구 사양 =
* WordPress 버전 3.5 이상.
* PHP version 5.2.4 이상.
* MySQL version 5.0 이상.

= 권장 요구 사항 =
* 워드프레스 최신 버전
* PHP version 5.4 이상.
* MySQL version 5.5 이상.

== Frequently Asked Questions ==

= 어떻게 심플3D 3D모델을 삽입하나요? =
플러그인이 실치된 후에는 포스트 편집기에서 원하는 모델의 URL을 붙여넣기만 하시면 됩니다. 

`http://simple3d.net/models/2bV`

= 삽입된 모델의 가로와 세로를 변경할 수 있나요? =
워드프레스 4.2부터는 삽입된 모델을 더블 클릭하면 최대 넓이와 최대 높이를 설정할 수 있습니다. 크기를 설정하면 워드프레스가 `[embed]` shortcode를 삽입하여 줍니다.
`[embed width="400" height="300"]http://simple3d.net/models/2bV[/embed]`
